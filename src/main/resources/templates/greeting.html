Hogwarts Houses

This a Harry Potter themed student and dormitory register application, where you are able to:

  • register students
  • create, edit and delete dormitories
  • assign students to dormitories
  • check the available dormitories
  • check the cat and owl free dormitories (for rat owners)

This project is my first Spring Boot web application. It was created during my studies, and it was one of the many mandatory mini project requested by the programming school (Codecool). The focus was on getting familiar with this framework. Other technologies like JavaScript, CSS, HTML are represented on a necessary minimum level. Data is stored in memory.
